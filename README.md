# Therapify

Therapify is a Flutter-based application designed to provide users with virtual therapy solutions. The app incorporates features such as live chat and voice communication with AI-powered therapists, session tracking, and nearby therapist recommendations.

## Features

- **Talk with Theera**: Start a voice session with an AI-powered virtual therapist.
- **Chat with Theera**: Engage in text-based therapy sessions with AI assistance.
- **Nearby Therapists**: View and connect with therapists in your vicinity.
- **User-Friendly Interface**: Intuitive and accessible design for seamless interaction.
- **Secure Storage**: All user data is encrypted and securely stored.

## Installation

1. **Clone the Repository**
   ```bash
   git clone https://github.com/Abhxjith/Therapify-App.git
   ```
2. **Navigate to the Project Directory**
   ```bash
   cd Therapify-App
   ```
3. **Install Dependencies**
   Ensure Flutter is installed, then run:
   ```bash
   flutter pub get
   ```

## Running the App

1. **Run on an Emulator or Physical Device**
   ```bash
   flutter run
   ```

2. **For iOS**
   - Make sure Xcode is installed and configured.
   - Use the command:
     ```bash
     flutter run
     ```

3. **For Android**
   - Ensure Android Studio is set up.
   - Use the command:
     ```bash
     flutter run
     ```
